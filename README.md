##Introduction of Document
The purpose of this system is to enable small organizations to manage their assets such as laptops, devices, tools, furniture, etc. The system ensures secure access through user roles (Admin, Manager, Viewer) and provides efficient features for asset tracking, updates, categorization, and reporting.
#Scope (of the project)

In-Scope Features
User Management
•	Role-based authentication (Admin, Manager, Viewer)
•	Secure login for all user types
•	Admin can create and manage user accounts

Asset Management
•	Add, view, update, and delete assets (Admin only for add/delete)
•	Track asset status (In Use, In Repair, Available, In Stock)
•	Assign or update asset locations (Admin, Manager)

Category & Location Management
•	Predefined list of categories
•	Admin can add, edit, delete categories
•	List of predefined locations (e.g., Office-A, Warehouse-1)

Search & Filtering
•	Search by name, serial number, or category
•	Filter by status, category, and location

Reports
•	Assets by Location
•	Low Stock Report (below threshold items)

Out-of-Scope Features
•	Real-time tracking using IoT devices
•	Integration with external inventory systems
•	Mobile application (responsive web only)
•	Automated procurement system
•	AI-based asset condition prediction
